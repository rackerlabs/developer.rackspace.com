Getting Started with .NET SDK

Rackspace's official OpenStack SDK for Microsoft .NET, OpenStack.NET, an MIT licensed open-source community project for using multiple cloud providers from a standardized API.

Currently, OpenStack.NET has support for multiple Rackspace services:

  • Auto Scale
  • Cloud Block Storage
  • Cloud DNS
  • Cloud Databases
  • Cloud Files
  • Cloud Load Balancers
  • Cloud Monitoring
  • Cloud Queues
  • Cloud Servers

Support for additional services is on-going.

Requirements

  1. Visual Studio 2008 or newer or SharpDevelop
  2. NuGet (Note, NuGet is included with Visual Studio 2010 or newer and SharpDevelop 4.1 or newer)

Installation

OpenStack.NET (and all of the necessary dependencies) are installed into your project using the NuGet Package Manager Console, using the following command:

{% highlight sh %} Install-Package openstack.net{% endhighlight %}

Using OpenStack.NET

Once an Identity has been created, you can specify the region for any method in the .NET SDK. If no region is specified in a method call, the default region will be used.

In this example, we create a single Identity and then output a list of server Id's in the IAD region to the console:

{% highlight csharp %} using net.openstack.Providers.Rackspace; CloudIdentity cloudIdentity = new CloudIdentity() { Username = "{username}", APIKey = "{apikey}" }; CloudIdentityProvider cloudIdentityProvider = new CloudIdentityProvider(cloudIdentity); Console.WriteLine("Getting list of your servers..."); CloudServersProvider csp = new CloudServersProvider(cloudIdentity); System.Collections.IEnumerable servers = csp.ListServersWithDetails(region: "IAD"); foreach (Server server in servers) { Console.WriteLine("Server ID: {0}", server.Id); } {% endhighlight %}
