Installing jclouds

Download this pom.xml file or put the dependencies section in your project's pom.xml. Run the Maven command below in the same directory as your pom.xml file.

{% highlight sh %} $ mvn dependency:copy-dependencies{% endhighlight %} {% highlight java %} import org.jclouds.ContextBuilder; import org.jclouds.openstack.nova.v2_0.NovaApi;{% endhighlight %}

Requirements

  1. Java Development Kit (JDK) version 6 or higher
  2. Maven version 3 or later
