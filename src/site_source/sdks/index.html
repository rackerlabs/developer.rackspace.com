Installing Gophercloud

Run the following command (after installing Go):

{% highlight sh %} go get github.com/rackspace/gophercloud {% endhighlight %}

Then use it in your code by importing it:

{% highlight go %} import ( "github.com/rackspace/gophercloud" "github.com/rackspace/gophercloud/rackspace" ) {% endhighlight %}

Requirements

  1. Go
  2. Git

Installing jclouds

Download this pom.xml file.

{% highlight sh %} mvn dependency:copy-dependencies "-DoutputDirectory=./lib" {% endhighlight %}

Tip: To make jclouds a part of your existing project, simply copy the relevant dependencies into your own pom.xml file.

Requirements

  1. Java Development Kit (JDK) version 6 or higher
  2. Maven version 3 or later
